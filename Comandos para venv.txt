En VS Code, abrí una terminal.
* Ejecutá:
python -m venv .venv

* Activá el entorno virtual:
.venv\Scripts\Activate

EN LINUX con GitBash (posicionada en la terminal de vsc, la flecha que está al lado del +, al lado de powershell)
-Levanto entorno virtual con: python -m venv venv
-Lo activo yendo a: directorio/Scripts y ejecutando activate (si es git bash)
-Vuelvo a mi carpeta con el entorno activado e instalo lo que necesito
-Para pasar a un documento mis dependencias instaladas: pip freeze > requirements.txt
